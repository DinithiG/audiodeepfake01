Audio Samples from "Interpretable and Controllable Singing Voice Decomposition via Assem-VC"

Paper: arXiv:21TBD

Repository: mindslab-ai/assem-vc @ GitHub

Authors: Kang-wook Kim, Junhyeok Lee @MINDsLab Inc., SNU

Abstract: We propose a singing decomposition system that encodes time-aligned linguistic content, pitch, and source speaker identity via Assem-VC. With decomposed speaker independent information and target speaker's embedding, we could synthesize singing voice of target speaker.
